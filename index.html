About the Contents

The images and data of the featured project contain information used to determine vacation spots according to weather conditions. APIs were used to obtain access to weather and maps from OpenWeatherMap and GoogleMaps. JSON was also used for this project. Additionally, Python was the tool chosen to manipulate the data and obtain the information required. While certain latitude and longitude points were used as shown in the thumbnail. The main focus for the contents are the weather items featured throughout the website.

About The Website

This is a visualization dashboard website using visualizations from past assignments, specifically Weather Data. Additionally, individual pages for each plot will be included along with a navigator. These pages will contain the visualizations and their corresponding explanations. The landing page will contain a comparison of all of the plots. A page containing the data used to build the rest of the pages will also be included.
